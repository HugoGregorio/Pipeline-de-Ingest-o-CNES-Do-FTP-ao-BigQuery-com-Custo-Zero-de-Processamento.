# Pipeline-de-Ingest-o-CNES-Do-FTP-ao-BigQuery-com-Custo-Zero-de-Processamento.
Pipeline ETL automatizado: do FTP do DATASUS (CNES) para o BigQuery via Python/Pandas. Resolve gargalos de rede, limpa schemas com Regex e modela a Camada Prata em memória. Arquitetura focada em FinOps (custo zero de transformação no DW) e SecOps (autenticação segura). Dados públicos prontos para análise e BI!
